# Broken Ring

The public home of **Broken Ring**, a Final Prime dark fantasy role-playing game.

This repository contains only reviewed material approved for public release. Draft lore, spoilers, internal design notes, AI playtest logs, benchmark data, and unreleased assets remain in the private Final Prime gameplay lab.

## Public site

The GitHub Pages build is deployed from [`public/`](public/) by [the Pages workflow](.github/workflows/pages.yml).

Live URL: <https://final-prime.github.io/Broken-Ring/>

## Release boundary

- `public/`: deployable public game build
- `PUBLIC_RELEASE_POLICY.md`: rules for moving material from the private lab into this repository
- `.github/workflows/pages.yml`: static Pages deployment

No private gameplay-lab content should be copied here without an explicit release review.
