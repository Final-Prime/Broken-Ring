# Public Release Policy

This repository is a curated publication target, not a development mirror.

## Allowed

- Approved playable builds
- Approved public lore and character material
- Public-facing artwork and audio with confirmed rights
- Release notes and player documentation
- Sanitized demo data required by the public build

## Not allowed

- Draft or alternate story branches
- Unreleased lore, endings, twists, or encounter solutions
- Dungeon Master notes and hidden state
- AI player transcripts, evaluations, prompts, or benchmark results
- Internal roadmaps, business material, security details, or credentials
- Assets without confirmed publication rights

## Release gate

Every transfer from the private gameplay lab must pass four checks:

1. Narrative review for spoilers and unfinished canon
2. Technical review for secrets, private paths, debug data, and source maps
3. Rights review for code, images, fonts, audio, and trademarks
4. Manual approval for public release

The private gameplay lab remains the source of ongoing development. This repository contains public snapshots only.
